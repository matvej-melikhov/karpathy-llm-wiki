# `benchmarking/` — замеры для главы 4

Папка содержит скрипты и сохранённые артефакты для четырёх количественных
замеров § 4.4 главы 4. Все замеры — **детерминированные** или
**теоретические оценки**: вместо живых сессий Claude Code сравнивается
реальная реализация vs гипотетическая стоимость LLM-альтернативы по
размеру входных и выходных артефактов в токенах.

## Сводная таблица (текущие числа на 62-страничном vault'е)

| Раздел | Замер | Реализовано | LLM-альтернатива | Выигрыш |
|---|---|---|---|---|
| § 4.4.1 | горячий кэш cache.md | 980 токенов | 12 417 токенов (log+index+4 стр.) | **12.7×** |
| § 4.4.2 | гибридный поиск | top-15 ≈ 8 521 токенов | index+5 стр ≈ 8 266 (на 62 стр.) → 33 281 (на 500 стр.) | от **1.03×** до **3.9×** |
| § 4.4.3 | Stop-hook регенерация | 2.71 с/turn, $0 | ~$9.29/turn LLM = $139/сессия | **∞** |
| § 4.4.4 | lint Layer 2 | $0.09 на проверку (3 пары) | $86.54 без L1.5 (1 891 пара) | **~945×** |

## Запуск всего

```bash
python3 benchmarking/summarize.py --rerun     # перезапустить всё + сводная таблица
python3 benchmarking/summarize.py --md        # markdown вместо ascii
```

Все скрипты идемпотентны, ничего в vault'е не меняют, на сети не зависят
(кроме `bench_query.py` — он вызывает локальный эмбеддер через
`bin/embed.py query`).

## Методологические заметки

- **Токены — tiktoken `cl100k_base`** (близко к токенайзеру Claude,
  погрешность ±5–10 % для русско-английского текста).
- **Цены — публичные тарифы Anthropic Claude Opus 4.7:** input $5/M,
  output $25/M, cache_write $6.25/M, cache_read $0.50/M. Если тарифы
  изменились — можно переопределить флагами или поправить константы.
- **LLM-альтернатива — теоретическая оценка**, не живой A/B. Каждый
  скрипт документирует своё допущение (FALLBACK_PAGES, типичное число
  страниц на query и т.п.). Это позволяет показать порядок выгоды без
  затрат на десятки сессий — direction is robust, точные числа конкретной
  сессии могут отличаться ±2×.
- **Версии:** `claude --version` = 2.1.143, Python 3.14, tiktoken 0.x.
